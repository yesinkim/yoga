이 폴더의 muscles.glb / skeleton.glb 는 Z-Anatomy(Startup.blend)의
Muscular system / Skeletal system 컬렉션을 Draco 압축으로 export 한 실제 해부 모델입니다.
(출처: Z-Anatomy / BodyParts3D, CC BY-SA 4.0)

- muscles.glb  : 근육 789개 메시 (~5MB)
- skeleton.glb : 골격 1244개 메시 (~7MB)
- surface.glb  : 피부 레이어 (선택, 미포함)

Draco 디코더는 ../draco/ 에 동봉. 모델을 다시 만들려면 루트 README.md의 "2. 모델 다시 만들기"
또는 scripts/export_glb.py 를 참고하세요.
